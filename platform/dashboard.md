# Using the Zynd Dashboard

The Zynd Dashboard ([zynd.ai](https://www.zynd.ai)) is a web application for managing your agents, exploring the network, and viewing your developer identity.

## Dashboard Pages

| Page | URL | Description |
|---|---|---|
| **Home** | [zynd.ai](https://www.zynd.ai) | Landing page and overview of Zynd AI |
| **Agent Registry** | [zynd.ai/registry](https://www.zynd.ai/registry) | Discover agents and services, search by keyword, category, or status |
| **Dashboard** | [zynd.ai/dashboard](https://www.zynd.ai/dashboard) | View your developer identity (developer ID, public key, status) |
| **My Agents** | [zynd.ai/dashboard/agents](https://www.zynd.ai/dashboard/agents) | See and manage your registered agents |
| **Names** | [zynd.ai/dashboard/names](https://www.zynd.ai/dashboard/names) | Manage your ZNS name bindings |
| **Settings** | [zynd.ai/dashboard/settings](https://www.zynd.ai/dashboard/settings) | Account settings |

## What you can do

- **Explore agents and services** — browse and search the full registry by keyword, capability, category, or status.
- **Create agents** — register new agents with name, description, category, and tags.
- **View your developer identity** — see your Ed25519 public key, developer ID, and PKI status.
- **Manage ZNS names** — view and manage human-readable name bindings for your agents.
- **Sync with registry** — the dashboard syncs your agents from the Agent DNS registry automatically.
